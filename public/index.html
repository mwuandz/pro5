Bước 1

Frontend builder

Kết nối AI thật

Backend gọi OpenAI Responses API để sinh JSON gợi ý template, bảng màu, nội dung và thứ tự section chuẩn hơn so với bản mô phỏng ở frontend.

Bước 3

Xuất bản / quản lý

Thêm đăng nhập, lưu database, custom domain, analytics và triển khai portfolio ra đường dẫn công khai như tên-user.yoursite.com.

FAQ

Một vài lưu ý về bản website này

Bản này đủ để bạn demo sản phẩm hoặc dùng làm nền tảng phát triển tiếp. Phần preview và export vẫn gọn nhẹ, còn AI thật được xử lý an toàn ở backend.

Website này có AI thật không?

Có. Trong bản này, nút “Tạo 3 mẫu bằng AI” gọi backend server-side để sinh gợi ý thật. Nếu chưa cấu hình API key, giao diện sẽ báo rõ và fallback về gợi ý local demo.

Website có tạo ra portfolio riêng cho người dùng không?

Có. Builder sẽ tạo một portfolio preview ngay trên màn hình, sau đó bạn có thể xuất file HTML riêng cho website giới thiệu bản thân của người dùng.

Có thể dùng để pitch dự án hoặc demo khách hàng không?

Có. Giao diện hiện tại đủ để trình bày ý tưởng sản phẩm, cho người khác nhập prompt, chọn template và tải website mẫu ngay lập tức.
